Installing DesignFolio+

  1. Upload the Theme - After downloading the theme file, you'll want to unzip it to the WordPress directory "/wp-content/themes" and upload it to your server.
  2. Activate the Theme - In your WordPress admin panel, go to 'Appearance > Themes' and select "DesignFolio+" as your new theme.
  3. Change the Logo - Simply visit the UpThemes Framework theme options page to change your logo and background stripe color or image.

Install Plugins

Optionally, you should install the following plugins to really make your installation of DesignFolio+ awesome.

Twitter Widget Pro

Twitter integration requires the installation of the Twitter Widget Pro plugin:

To install and activate in your theme:

  1. Go to 'Plugins > Add New'
  2. Search for 'Twitter Widget Pro'
  3. Click the 'install' link to the right of the TWitter Widget Pro plugin.
  4. Continue by clicking the 'install' button.
  5. Enter your FTP connection information and click 'Proceed.'
  6. If successful, click the 'Activate' button.
  7. Go to 'Appearance > Widgets > Twitter Widget Pro'
  8. Click and drag the 'Twitter Feed' widget under the "Above Content - Homepage" widget area.
  9. Enter your twitter username and set up the options and number of tweets to display on each page.
  10. Click the "Save' button. You're done! You should now see twitter updates on the homepage.

Breadcrumb NavXT

The DesignFolio+ theme supports custom Breadcrumbs using the 'Breadcrumb NavXT' plugin.

To install, follow the process below:

  1. Go to 'Plugins > Add New'
  2. Search for 'Breadcrumb NavXT'
  3. Click the 'install' link to the right of the Breadcrumb NavXT plugin.
  4. Continue by clicking the 'install' button.
  5. Enter your FTP connection information and click 'Proceed.'
  6. If successful, click the 'Activate' button.

Once activated, your breadcrumbs should appear on each page.

Additional Instructions

Setting up your Portfolio

One of the best things about this theme is its ability to hold a nice portfolio section. Most of this is handled automatically, though there are a few things to know.

  1. Create a category called 'Portfolio' (this is the most crucial step, the theme will automatically detect this category and pull the first image from every post within this category in the homepage slider.
  2. Make a new post for each item in your Portfolio.
  3. For each post, you should use the WYSIWYG editor to upload and insert the image you want to display in the homepage slider. You can add text and other images within the post, but the homepage slider will ONLY SHOW the first image inserted in each post. IMPORTANT: The size of the image should ALWAYS be 580x350 for each item. You can leave white space inside this area but for the image slider to function properly the images must be the correct size.
  4. Enjoy your awesome and easy-to-use Portfolio section.
